# My Frontend Study Journey

This repository documents my learning journey in frontend development.

Instead of following traditional tutorials step-by-step, I train using a task-based approach. I use AI as a mentor that acts like a **senior developer or a client**. I give the AI a topic (for example: responsive cards, typography layout, or image gallery), and it provides a detailed project brief. I then build the project independently using the technologies I have already learned.

The goal of this repository is to show my **progress, problem-solving process, and practical implementation of frontend concepts.**

## Learning Method

My workflow usually looks like this:

1. I define a topic for the day (for example: responsive cards or typography layout).
2. AI acts as a **senior developer or client** and gives me a project brief.
3. I build the project completely by myself.
4. The finished project is added to this repository.

Some projects take **a few days**, while others take **one or two weeks**, depending on complexity.
